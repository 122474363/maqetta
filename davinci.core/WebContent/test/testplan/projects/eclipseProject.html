Eclipse project basic tests

Pull down the most recent Maqetta build

Steps

  1. Start maqetta in mutli user mode. (repeat tests in single user mode)
  2. Create a new project by going to New-->Project in the menu bar.
  3. Type a name for the project, and click the "Enable Eclipse Support"
  4. Create a new folder under WebContent by going to the New-->Folder in the title bar. Call the folder aaa
  5. Create a new HTML file in the WebContent/aaa folder by going to New-->HTML File in the menu bar
  6. Add some content to the new HTML file. Save and close
  7. Download the entire workspace as a zip by clicking the "Download Workspace" icon on the File Explore shortcut bar.
  8. Default settings are fine, click OK and save the file somewhere on the file system
  9. Start Eclipse. Then go to your workspace and right click and select Import-->Import
  10. In "General"" select "Existing Project into Workspace", Click the radio button next to "From Archive" then browse to find the zip file you just downloaded.
  11. The Maqetta project should show up in the lists of projects. Select it and click OK

Expected results

You should now have a project in your Eclipse workspace with the same name and structure as your maqetta project. You should be able to open any of the HTML files in a browser or editor.
