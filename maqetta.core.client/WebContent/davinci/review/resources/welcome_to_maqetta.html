Welcome to Maqetta Review and Commenting
Welcome to Maqetta's Review and Commenting module. You arrived here because you were invited to a review session, probably via an invitation email.
How to start a review session and provide feedback
To start a review session and provide feedback, look at the Reviews palette and find the appropriate review activity. (Note: there might only be review activity in the Reviews palette.) Click on the "+" icon to open up that review activity and show the list of files that you are to review. If you double-click on one of those files, that file will open in the main canvas (obscuring this welcome page), at which point you can add comments and annotations using the Comments palette and switch among different interactive states in the States palette.
