Theme sets test plan

Add new theme set.

Steps

  1. Choose New Theme from the main menu.
  2. Select claro from the drop down list Theme to clone.
  3. Enter claro2 in the new name box.
  4. When the theme editor is displayed select the global widget and change the background color. Save the theme and close the theme editor.
  5. Choose New Theme from the New menu.
  6. Select custom from the drop down list Theme to clone.
  7. Enter custom2 in the new name box.
  8. When the theme editor is displayed select the global widgets and change the background color. Save the theme and close the theme editor.
  9. Choose settings->Theme sets from the main menu.
  10. From the Manage theme sets dialog, click on the below the Theme sets: selection list. A theme set called custom_default_1 Should be added to the list of themes.
  11. With the custom_default theme set still selected in the Theme sets: selection list click on the + a theme set called custom_default_2 should be added to the list.
  12. With the custom_default_2 theme set still selected int the Theme sets: selection list click on the + a theme set called custom_default_2_1 should be added to the list.
  13. Click Ok.
  14. Reopen Manage theme sets Settings -> Theme sets

Expected results

  1. Theme sets: selection list should have the following theme sets.
    1. custom_default
    2. custom_default_1
    3. custom_default_2
    4. custom_default_2_1

Remove theme sets unsaved

Steps

  1. Continuing from the test case above, with the Manage theme sets dialog displayed.
  2. Select custom_default_2 in the Theme sets: selection list.
  3. Click on the "-" below the Theme set: selection list, custom_default_2 should be removed from the list.
  4. Select custom_default_2_1 in the Theme sets: selection list.
  5. Click on the "-" below the Theme set: selection list, custom_default_2_1 should be removed from the list.
  6. Click Cancel button.
  7. Reopen Manage theme sets Settings -> Theme sets

Expected results

  1. Theme sets: selection list should have the following theme sets.
    1. custom_default
    2. custom_default_1
    3. custom_default_2
    4. custom_default_2_1
